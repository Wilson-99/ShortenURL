Projeto de Encurtamento de URL com QR Code

Este é um projeto simples que permite aos usuários encurtar URLs longas e gerar códigos QR para essas URLs. O projeto é construído usando React.js e integra a API do Bitly para encurtar URLs.
Funcionalidades

    Encurtar URL: Os usuários podem inserir uma URL longa e clicar no botão "Encurtar" para obter a URL encurtada.
    Gerar QR Code: Os usuários podem optar por gerar um código QR para a URL encurtada.
    Validação de URL: A aplicação realiza uma validação simples da URL para garantir que ela tenha o formato correto.
    Compartilhamento em Redes Sociais: Os usuários podem compartilhar a URL encurtada em redes sociais como Facebook, Twitter, WhatsApp, Instagram e LinkedIn.

Configuração

Antes de executar o projeto, é necessário configurar a chave de API do Bitly. Substitua o valor de bitlyAPIKey no arquivo Home.jsx com a sua chave de API do Bitly.
Instalação

    Clone o repositório:

bash

git clone https://github.com/seu-usuario/nome-do-projeto.git

    Navegue até a pasta do projeto:

bash

cd nome-do-projeto

    Instale as dependências:

bash

npm install

    Inicie o servidor de desenvolvimento:

bash

npm run dev

Uso

    Insira a URL longa no campo de entrada.
    Clique no botão "Encurtar" para obter a URL encurtada.
    Marque a opção "Gerar QR Code" se desejar um código QR para a URL encurtada.
    Compartilhe a URL encurtada em redes sociais usando os botões de compartilhamento.
